# jokersgambit
A fast-paced, high-stakes 2-player card game built with raylib* Quick Overview   Single-screen competitive card game for 1 or 2 human players (or watch AI vs AI).   Goal: be the first to complete **3 out of 5** target ranks (A♡ K♡ Q♡ J♡ 10♡)  First to 3 completed ranks wins and gets their score immortalised on the leaderboard
